# Faculdade
Repositório da Faculdade de Ciência da Computação


📚 Repositório de Trabalhos da Faculdade

Este repositório tem como objetivo organizar e centralizar todos os meus trabalhos acadêmicos desenvolvidos durante o curso de Ciência da Computação.

Os materiais estão estruturados de forma clara, separados por período e, dentro de cada um, organizados por matéria, facilitando o acompanhamento da minha evolução ao longo da graduação.

📂 Organização

Cada período contém suas respectivas disciplinas, com atividades, projetos, listas de exercícios e trabalhos práticos desenvolvidos ao longo das aulas
🎯 Objetivo
Documentar meu progresso acadêmico
Facilitar revisão de conteúdos
Servir como portfólio de aprendizado
Acompanhar minha evolução técnica ao longo do curso
